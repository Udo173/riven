## Content

**Ways of archiving projects**

Depending on the nature of the project, PMs might select to archive files digitally in cloud-based storage or physically in cabinets. While archiving, PMs apply the following guidelines:

- Save and store digital files to external hard drives or backup servers.
- Use an organized filing system with labels and cabinets for printed materials.
- Use project management software tools for convenience and efficiency.
- Use standard templates and checklists for consistency and completeness.
- Create a detailed project closure report with key information, lessons learned, and outcomes.
- Archive important email communications related to the project in a format that allows easy search and retrieval.

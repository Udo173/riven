# MONITOR AND CONTROL PHASE

---

## ORIGINAL TEXT:

**The importance of monitoring and controlling a project**
PMs complete the monitor and control phase with the project's execution phase. This phase serves as the project's guidance system where PMs:

- Track actual performance against the baseline plan
- Identify and analyze deviations
- Manage changes to project scope, schedule, and budget
- Implement corrective actions to address issues
- Report progress to stakeholders

Without proper monitoring and control, even well-planned and executed projects can result in missed deadlines, budget overruns, and deliverables that don't meet stakeholder expectations. So, PMs continuously monitor and control project activities to identify potential problems early, make informed decisions, and successfully complete the project.

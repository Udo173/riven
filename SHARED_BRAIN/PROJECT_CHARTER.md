## PROJECT CANVAS (Alternative Darstellung)

### Definition:
> Ein Project Canvas ist eine visuelle Übersicht aller Projekt-Komponenten auf einen Blick.

### Erklärung:
> Es erfasst alle wichtigen Elemente des Project Charters (Ziele, Scope, Stakeholder, Meilensteine) in einer kompakten, visuellen Darstellung. Perfekt als Referenz für das Team.

### Nutzen:
> Hilft fokussiert zu bleiben, Fortschritt zu monitoren, neue Mitglieder einzubinden, und am Ende Lessons Learned zu identifizieren.

### Canvas Elemente:
| Element | Beschreibung |
|---------|-------------|
| Goals | Projektziele |
| Users | Zielgruppe/Nutzer |
| User Benefits | Nutzen für Anwender |
| Team | Projektteam |
| Stakeholders | Beteiligte Parteien |
| Scope | Projektumfang |
| Deliverables | Liefergegenstände |
| Activities | Aktivitäten/Tasks |
| Milestones | Meilensteine |
| Risks | Risiken |

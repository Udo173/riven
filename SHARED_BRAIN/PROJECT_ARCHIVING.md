# Ways of Archiving Projects

## Overview
Depending on the nature of the project, PMs might select to archive files digitally in cloud-based storage or physically in cabinets.

## Guidelines

### 1. Digital Archive
- Save and store digital files to external hard drives or backup servers
- Use cloud-based storage for accessibility

### 2. Physical Archive
- Use an organized filing system with labels and cabinets
- Maintain proper physical organization

### 3. Project Management Software
- Use PM software tools for convenience and efficiency
- Centralize project documentation

### 4. Templates & Checklists
- Use standard templates for consistency
- Use checklists for completeness

### 5. Project Closure Report
- Create detailed closure report with:
  - Key information
  - Lessons learned
  - Outcomes
  - Success metrics

### 6. Email Communication
- Archive important emails related to project
- Use format that allows easy search and retrieval

## Key Takeaways

| Methode | Vorteil | Tool/Format |
|---------|---------|-------------|
| Digital | Zugriff von überall | Cloud/HDD |
| Physical | Sicherheit | Schränke/Labels |
| PM Software | Zentralisiert | Tools |
| Templates | Konsistenz | Standardisiert |
| Closure Report | Lernen | Dokumentiert |
| Email | Nachvollziehbar | Durchsuchbar |

## Checkliste

```
□ Project Name definiert
□ Project Description klar
□ Project Purpose formuliert
□ Deliverables aufgelistet
□ Budget gesetzt
□ Risks & Constraints identifiziert
□ Milestones mit Daten
□ Messbare Objectives
□ Key Stakeholder bekannt
□ PM und Sponsor benannt
□ Status gesetzt
```

# PM KEY CONCEPTS - Planung Phasen

---

## ORIGINAL TEXT:

**Key concepts**

**1.** During the planning part of the initiate and plan a project phase, project managers take the following steps to create a realistic plan:

- Step 1: Create a work breakdown structure (WBS)
- Step 2: Estimate effort and duration
- Step 3: Create the project schedule
- Step 4: Estimate cost
- Step 5: Adjust and finalize the baseline plan

**2.** A WBS is a hierarchical breakdown of a project's scope into smaller, more manageable tasks. To create a WBS, PMs can use a numbered list or diagram.

**3.** Effort is the total amount of work needed to complete a task and duration refers to the total time it takes to complete a task. To accurately estimate effort and duration, project managers consider the team size, task complexity, and dependencies.

**4.** A project schedule is a structured timeline that outlines tasks, durations, dependencies, and key milestones. To build a project schedule, PMs follow a structured approach that includes the following actions: Set start dates, Set finish dates, Identify dependencies.

**5.** To effectively organize a project schedule, PMs create a Gantt chart that serves as a visual timeline for all tasks and their details.

**6.** Project costs refer to the direct, indirect, fixed, and variable expenses that are required to complete a project. To accurately estimate the project costs, PMs consider the following cost categories: resource cost, materials cost, contingency fund, and total project cost.

**7.** A baseline plan is the finalized project plan, including scope, budget, and timeline, that serves as the official guide for the project. PMs finalize the baseline plan after receiving approval from teams and stakeholders.

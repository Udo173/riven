# Riven 🎵

**Udo Sassik's Musik-Projekt - Lyric Videos & Motion Graphics**

## Über Riven

Riven ist Udo Sassik's musikalisches Projekt. Der Name "Riven" bedeutet "gespalten" oder "zerrissen" - ein kraftvolles Symbol für den Kampf des Aufstehens.

## Songs

### Rise Every Time
- **Dauer:** 5:29 min
- **Genre:** Motivational/Electronic
- **Lyrics:** Inspiriert vom Kampf, niemals aufzugeben

## Design System

Dieses Repository nutzt professionelle Design-Systeme inspiriert von:
- **Linear** - Dark-mode-first Ästhetik
- **Stripe** - Premium Fintech Design
- **Vercel** - Developer-focused Minimalismus
- **Framer** - Cinematische Motion

Siehe [DESIGN.md](./DESIGN.md) für das vollständige Design-System.
